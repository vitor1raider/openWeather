## 📝 Objetivo

Desenvolver uma aplicação de previsão do tempo com base na API OpenWeather.

## 📌 Desenvolvimento

Para o desenvolvimento do projeto, foi utilizada a API do OpenWeather, com informações como:

- Cidade
- Temperatura
- Umidade
- Pressão
